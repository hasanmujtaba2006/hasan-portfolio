# 🌐 Hasan Mujtaba | Interactive 3D Developer Portfolio

A modern, highly interactive web portfolio showcasing my projects across Full-Stack Web Development, Python architecture, and the Internet of Things (IoT). 

Built with an emphasis on immersive UI/UX, this portfolio features a dark "glassmorphism" aesthetic, butter-smooth scroll physics, and live 3D rendering.

![Portfolio Preview](https://via.placeholder.com/1000x500/0a0a0a/34d399?text=Interactive+3D+Developer+Portfolio)

## ✨ Key Features

* **Live 3D Rendering:** Integrates a fully interactive, draggable 3D Spline scene right in the hero section.
* **Custom Spatial Cursor:** A custom-built, physics-based glowing emerald cursor with a trailing ambient light effect.
* **Smart Media Cards:** Project cards that dynamically render looping video demonstrations.
* **Terminal Typewriter Fallback:** If a project lacks video, the card seamlessly falls back to a custom-built, animated CLI terminal detailing the project execution.
* **Scroll Physics:** Elements smoothly fade and glide into view using Framer Motion viewport triggers.
* **Glassmorphism UI:** Frosted glass navigation and project cards that beautifully capture the light from the custom cursor.

## 🛠️ Tech Stack

* **Framework:** React + Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **3D Engine:** Spline (@splinetool/react-spline)
* **Deployment:** Vercel

## 💻 Run Locally

To clone and run this portfolio on your local machine:

```bash
# Clone the repository
git clone [https://github.com/yourusername/hasan-portfolio.git](https://github.com/yourusername/hasan-portfolio.git)

# Navigate into the directory
cd hasan-portfolio

# Install dependencies
npm install

# Start the local Vite development server
npm run dev
